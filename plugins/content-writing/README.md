# Content Writing Plugin

A content writing plugin that helps you plan, structure, and refine compelling presentations and slide decks.

## Components

### Skills

**slide-outline** — Guides you through a complete 5-step process for building presentation outlines: setup → key message → storyboard → draft → polish. Draws on BCG LAB slide writing principles for professional-quality decks.

## Usage

### slide-outline skill

Trigger this skill by saying things like:
- "Help me structure my presentation"
- "Create a slide outline for..."
- "Storyboard my deck"
- "Plan my slides"
- "Help me tell a story with slides"
- "What slides should I include?"

The skill walks through five interactive steps:
1. **Setup** — Identify format, audience, and goal
2. **Key Message** — Build a 3-level argument tree
3. **Storyboard** — Translate the tree into a slide-by-slide outline
4. **Draft** — Write each slide using the DDR process (Draft → Drain → Refine)
5. **Polish** — Final review across story, slide-level, and language checks

## Setup

No environment variables or external configuration required. This plugin works entirely within your conversation.
